## ანალიზი და ხედვა

შენი ხედვა ძალიან ლოგიკურია - მთავარი გვერდი უნდა იყოს "ტრაფიკის განაწილების ცენტრი", სადაც მომხმარებელი სწრაფად პოულობს სწორ მიმართულებას. ამჟამინდელი გვერდი საკმაოდ გადატვირთულია და ბევრი ღირებული ფუნქცია (ბენზინის სადგურები, ელექტრო სადგურები, VIP სერვისები) საერთოდ არ ჩანს.

## ძირითადი ცვლილებები

### 1. Hero სექციის გამარტივება
**მიზანი:** ძიების პროცესის გამარტივება

**ცვლილებები:**
- ძიების ფორმის გამარტივება - მხოლოდ ტექსტური ძიება + კატეგორია + ქალაქი
- "ავტომობილის მარკები" გადავა ცალკე გვერდზე
- ევაკუატორის ღილაკი დარჩება, მაგრამ უფრო კომპაქტური

### 2. კატეგორიების კარუსელი (ახალი კომპონენტი)
**ფაილი:** `src/components/home/CategoryCarousel.tsx`

**დიზაინი:**
- ჰორიზონტალური სქროლი `embla-carousel`-ით
- თითოეული კატეგორია: მრგვალი აიკონი + სახელი
- პოპულარული კატეგორიები პირველი (მეხსიერების მიხედვით: popularity-based sorting)
- მობაილზე: touch-friendly swipe
- დესკტოპზე: ისრები მარცხნივ/მარჯვნივ

## ტექნიკური დეტალები

### ახალი კომპონენტების სია

| ფაილი | აღწერა |
|-------|--------|
| `src/components/home/CategoryCarousel.tsx` | კატეგორიების ჰორიზონტალური კარუსელი |
| `src/components/home/VIPServicesCarousel.tsx` | VIP სერვისების კარუსელი |
| `src/components/home/StationsPromo.tsx` | საწვავი/ელექტრო სადგურების პრომო სექცია |
| `src/components/home/SimplifiedSearch.tsx` | გამარტივებული ძიების ფორმა |

### გამოყენებული ბიბლიოთეკები
- `embla-carousel-react` - უკვე დაინსტალირებულია პროექტში
- Existing hooks: `useFuelStations`, `useChargers`, service fetching

### მონაცემთა ბაზის query-ები

**VIP სერვისების fetch:**
```sql
SELECT * FROM mechanic_services 
WHERE vip_status IS NOT NULL 
AND is_vip_active = true 
ORDER BY 
  CASE vip_status WHEN 'super_vip' THEN 1 WHEN 'vip' THEN 2 END
LIMIT 10
```

**კატეგორიების fetch:** უკვე არსებული query გამოყენება priority-based sorting-ით

### Index.tsx ცვლილებები

**წაშლა:**
- რთული ModernServiceFilters კომპონენტი Hero-დან
- Quick Access ბარათები (ინტეგრირდება სხვაგან)
- კატეგორიების grid სექცია (ჩანაცვლდება კარუსელით)

**დამატება:**
- CategoryCarousel კომპონენტი
- VIPServicesCarousel კომპონენტი  
- StationsPromo კომპონენტი
- SimplifiedSearch კომპონენტი
